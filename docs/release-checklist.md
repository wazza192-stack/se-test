# Release Checklist

## Responsive checks

- [ ] Page works at 375px width
- [ ] Page works at 768px width
- [ ] Page works at 1024px width
- [ ] Page works on desktop width

## Layout checks

- [ ] No unwanted horizontal scrolling
- [ ] No overlapping elements
- [ ] No clipped text or cut-off buttons
- [ ] Spacing looks consistent
- [ ] Sections stack correctly on smaller screens
- [ ] Cards and panels resize cleanly

## Typography checks

- [ ] Body text is readable on mobile
- [ ] Headings scale cleanly
- [ ] Text does not overflow its container
- [ ] Line breaks look natural

## Navigation checks

- [ ] Navigation works on mobile
- [ ] Menus open and close properly
- [ ] Important links remain accessible
- [ ] Sticky elements do not block too much screen space

## Forms and controls

- [ ] Buttons are easy to tap
- [ ] Inputs are readable and usable
- [ ] Dropdowns and selectors work on mobile
- [ ] Validation messages are visible and understandable
- [ ] Forms can be completed on a phone without layout issues

## Media and content

- [ ] Images scale correctly
- [ ] Videos or embeds fit the screen
- [ ] Tables have a usable mobile solution
- [ ] Key content is present on mobile as well as desktop

## Final review

- [ ] Mobile experience feels intentional, not compressed desktop
- [ ] Desktop improvements have not broken mobile
- [ ] Similar components behave consistently
- [ ] No obvious responsive regressions introduced in this release
